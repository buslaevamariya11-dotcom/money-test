# Money-Test

![Java CI](https://github.com/buslaevamariya11-dotcom/money-test/actions/workflows/gradle.yml/badge.svg)

Автотесты для веб-приложения **iBank**, предназначенного для перевода денежных средств между банковскими картами
пользователя.

## Описание проекта

Проект реализует UI-автотестирование банковского веб-приложения с использованием паттерна **Page Object**.

Покрыты следующие сценарии:

- логин пользователя
- подтверждение входа кодом
- отображение балансов карт
- перевод средств между картами
- негативный сценарий перевода сверх доступного баланса

## Используемые технологии

- **Java 11**
- **Gradle**
- **JUnit 5**
- **Selenide**
- **Lombok**
- **GitHub Actions (CI)**

## Реализованные тесты

### Перевод между картами

Проверяется корректность перевода допустимой суммы:

- баланс карты отправителя уменьшается
- баланс карты получателя увеличивается

Статус: **PASSED**

### Негативный сценарий — перевод сверх баланса

Проверяется запрет перевода суммы, превышающей остаток на карте.

Ожидаемое поведение:

- операция не выполняется
- балансы остаются без изменений

Фактическое поведение:

- баланс изменяется
- приложение позволяет перевод сверх остатка

Статус: **FAILED (фиксирует дефект)**

## Обнаруженный дефект

В приложении отсутствует корректная проверка превышения доступного баланса карты отправителя.

Это приводит к возможности перевода суммы больше остатка.

## Запуск приложения

Перед запуском тестов необходимо запустить тестовое приложение:

java -jar artifacts/app-ibank-build-for-testers.jar

Приложение доступно по адресу: http://localhost:9999

Запуск тестов: Локально ./gradlew clean test
